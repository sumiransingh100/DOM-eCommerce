# DOM eCommerce 🛒

A lightweight, fully functional e-commerce application built with vanilla JavaScript, HTML, and CSS. This project demonstrates DOM manipulation, dynamic content rendering, and local storage persistence without any external frameworks or libraries.

## 🎯 Features

- **Product Management**: Create, read, update, and delete products with a clean interface
- **Persistent Storage**: All products are saved to browser localStorage for data persistence
- **Pre-loaded Data**: Comes with 8 default electronics products for immediate demonstration
- **Dynamic Rendering**: Products are rendered dynamically as cards with images, descriptions, and prices
- **Modal Forms**: User-friendly form modal for creating new products
- **Responsive Design**: Modern, gradient-based UI with smooth animations and hover effects
- **Product Actions**: Update product details or delete products with simple button clicks

## 🛠️ Tech Stack

- **HTML5** - Semantic markup and structure
- **CSS3** - Modern styling with gradients, animations, and flexbox layout
- **JavaScript (Vanilla)** - DOM manipulation, event handling, and localStorage API
- **LocalStorage API** - Client-side data persistence
